# To Compile
`lilypond [-o <output-path>] <source-ly-filepath>`

Example:
`/opt/lilypond/lilypond-2.24.4/bin/lilypond -o ./dwarf-fortress-theme ./dwarf-fortress-theme.ly`

See: https://lilypond.org/doc/v2.24/Documentation/usage/command_002dline-usage
